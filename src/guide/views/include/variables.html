box shadow

Example Variables Description
var(--shadow-sm) 버튼 등 크기가 작은 요소에 사용.
var(--shadow-md) 일반적으로 가장 많이 사용. dropdown, modal 등
var(--shadow-lg) 가장 상위에 있거나, 가장 강조하고 싶거나, 크기가 큰 레이아웃에 사용

border radius

Example Variables Value Description
var(--border-radius-sm) 4px
var(--border-radius-md) 6px
var(--border-radius-lg) 8px 드롭다운 등 form요소보다 큰 요소에 사용
var(--border-radius-max) 30px chip에 사용
var(--border-radius-circle) 50% 원형 badge에 사용
var(--form-radius) 6px form요소(input, button, select, textarea..)에 사용
