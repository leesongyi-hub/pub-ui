참고문서 바로가기 		<사용방법>
0. node.js 설치
1. VSCODE 터미널에서 npm으로 설치
npm install vue@2
npm install vuejs-datepicker --save

2. HTML파일에 연결
script src="../../node_modules/vue/dist/vue.js"
script src="../../node_modules/vuejs-datepicker/dist/vuejs-datepicker.js"

SCSS 파일구조

*CSS파일, SCSS파일은 퍼블리셔만 수정 가능. SCSS 모든 파일은 최종 style.CSS로 컴파일되어 각 html에 link

*불가피하게 개발에서 바로 CSS추가/수정이 필요하다면 별도로 CSS파일(ex)dev.css) 추가하여 html에 link
